Папка media — фотографии для сайта RiverFish70
==============================================

1) Ручное копирование
   Положите файлы сюда и в конструкторе укажите имя файла или путь media/имя.jpg

2) Через конструктор (cms.html) при работе по HTTP
   Откройте cms.html через PHP (например http://localhost/.../cms.html), вкладка «Папка фото»:
   — список файлов из этой папки;
   — загрузка с компьютера;
   — удаление;
   — кнопка «Копировать путь» для вставки в товар или блок «Первый экран».

   Для этого рядом должен лежать файл media-api.php и быть включён PHP.
   Пароль тот же, что в config.php для CMS.

3) Если открыть cms.html двойным щелчком с диска (file://)
   браузер не даст управлять файлами на сервере — используйте URL с http:// или cms.php на хостинге.

4) VS Code «Live Server» (порт 5500 и похожие)
   Он отдаёт файлы как статику и НЕ запускает PHP: media-api.php в браузере будет виден как текст <?php ... — загрузка фото из конструктора не заработает.
   Запустите в корне сайта start-php-server.cmd (нужен PHP в PATH) и откройте http://127.0.0.1:8080/cms.html
   либо OpenServer / XAMPP.
